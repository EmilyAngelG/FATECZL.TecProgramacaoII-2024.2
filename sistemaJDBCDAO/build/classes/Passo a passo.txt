DA ESTRUTURA DO PROJETO
Para conectar o Java ao banco de dados utilizando o padrão DAO precisaremos de três partições que separaremos em pacotes:

-> br.com.fatec.dao

   Vale resaltar que estamos aprendendo a estabelecer a conexão com banco de dados relacional, por hora não veremos banco de dados não relacional em Java
